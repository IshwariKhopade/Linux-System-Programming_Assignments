# Linux System Programming Assignments

This repository contains Linux system programming assignments and example codes developed by **Ishwari B. Khopade**. The codes follow industrial coding conventions and include beginner-level programs.

## Features
- Beginner-friendly Linux system programming examples
- Follows industry-standard coding conventions
- Covers multiple assignments

## Prerequisites
- Linux operating system
- GCC compiler installed

## How to Compile & Run

### For C programs:
1. Open a terminal and navigate to the directory containing the C file.
2. Compile the program using GCC:

   gcc Assignment1_1.c -o myexe

3. ./myexe
